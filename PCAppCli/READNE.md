# ResponsiveAppCli

## description
   此项目是一个用于 PC 端的 Web 应用，为了兼容 IE9，包含基础功能和自定义的脚手架。

## 技术栈:
   Vue.js2
   Vuex
   Vue Router
   Axios
   Element UI
   Babel (兼容 IE9)
   @babel/preset-env
   babel-polyfill
   scss
   Webpack

   
## 克隆项目
git clone https://github.com/yourusername/PCAppCli.git

## 进入项目目录
cd MyPCApp

## 安装依赖
npm install

## 运行项目
npm run serve
